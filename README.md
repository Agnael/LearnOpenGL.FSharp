# LearnOpenGL.FSharp
Exercises based on the https://learnopengl.com/ tutorials, using F# and SILK.NET as the OpenGL wrapper.
Each title title with a link is pointing to the original chapter from the learnopengl.com site on which the theory is explained and all of the math is being copied from.

## [Transformations](https://learnopengl.com/Getting-started/Transformations) 
#### 07_Transformations
This one looks kind of weird because it the images are not centered vertically, the intention is to be forced to realize that rotations in OpenGL are made from the origin and if a rotation of an object is needed around itself, then said object must be translated to the origin, rotated and then translated back to it's initial position, which may be somewhat unintuitive.
